문자열 `code`가 주어질 때, 길이가 4보다 짧으면 앞을 `0`으로 채워 정확히 4자리 이상이 되도록 만든 문자열을 반환하는 `solution` 함수를 작성하세요.

## 오늘의 메서드
`padStart()`는 문자열의 앞쪽을 원하는 문자로 채워 길이를 맞출 때 사용하는 메서드입니다.

## 메서드 설명
이 문제에서는 주문 코드가 너무 짧을 때 앞에 `0`을 붙여 보기 좋은 형식으로 맞춰야 합니다.
`padStart()`를 사용하면 반복문 없이도 문자열 길이를 손쉽게 맞출 수 있습니다.

예를 들어 `"7".padStart(4, '0')`은 `"0007"`을 반환합니다.
이미 길이가 4 이상이면 원래 문자열을 그대로 반환합니다.

## 기본 문법
```js
str.padStart(targetLength, padString)
```

## 사용 예시
```js
'7'.padStart(4, '0') // '0007'
'42'.padStart(4, '0') // '0042'
'1234'.padStart(4, '0') // '1234'
```

## 주의할 점
- `padStart()`는 **문자열 메서드**입니다. 숫자라면 먼저 문자열로 바꿔야 합니다.
- 원본 문자열을 직접 바꾸는 것이 아니라, **새 문자열을 반환**합니다.
- 목표 길이보다 이미 길면 잘라내지 않고 그대로 둡니다.

## 제한사항
- `code`는 숫자로만 이루어진 문자열이라고 가정합니다.
- `code`의 길이는 1 이상 20 이하입니다.
- 반환값은 앞을 `0`으로 채운 문자열입니다.

## 예시
- 입력: `"7"` → 출력: `"0007"`
- 입력: `"42"` → 출력: `"0042"`
- 입력: `"1234"` → 출력: `"1234"`

## 힌트
- 문자열 길이를 4로 맞추고 싶다면, 앞을 채우는 메서드를 떠올려 보세요.
- 길이가 이미 충분한 경우에도 같은 메서드로 처리할 수 있습니다.

## 해설
이 문제의 핵심은 **문자열 길이가 부족할 때만 앞에 `0`을 채우는 것**입니다.

`padStart(4, '0')`를 사용하면 문자열의 전체 길이가 4가 될 때까지 앞쪽에 `0`을 붙입니다.
그래서:

1. `"7"`은 `"0007"`이 됩니다.
2. `"42"`는 `"0042"`가 됩니다.
3. `"1234"`는 이미 길이가 4이므로 그대로 `"1234"`가 됩니다.
4. `"98765"`처럼 더 길어도 자르지 않고 그대로 반환합니다.

이 문제를 통해 `padStart()`가 **코드 번호, 날짜, 시간, 일련번호처럼 자릿수를 맞추는 작업**에 자주 쓰인다는 점도 함께 익힐 수 있습니다.
