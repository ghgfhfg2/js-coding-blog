문자열 `s`가 주어질 때, **왼쪽부터 봤을 때 처음으로 한 번만 등장하는 문자**를 반환하는 `solution` 함수를 작성하세요.

그런 문자가 없다면 빈 문자열 `""`을 반환하세요.

## 제한사항
- `s`는 영문 소문자로만 이루어진 문자열입니다.
- `0 <= s.length <= 100000`
- 반환값은 길이 1의 문자열 또는 빈 문자열 `""`입니다.
- 대소문자 구분은 없으며, 입력은 모두 소문자라고 가정합니다.

## 예시
- 입력: `"swiss"` → 출력: `"w"`
  - `s`는 여러 번 나오고, `w`는 한 번만 등장하면서 가장 먼저 나옵니다.
- 입력: `"aabbccd"` → 출력: `"d"`
- 입력: `"aabbcc"` → 출력: `""`

## 힌트
- 어떤 문자가 몇 번 나왔는지 먼저 세어 두면 편합니다.
- 그다음 문자열을 다시 처음부터 보면서, 등장 횟수가 `1`인 문자를 찾으면 됩니다.
- 한 번만 등장한 문자가 아예 없을 수도 있습니다.

## 해설
이 문제의 핵심은 **등장 횟수**와 **원래 순서**를 함께 보는 것입니다.

예를 들어 `"swiss"`에서 한 번만 등장한 문자는 `w`, `i` 두 개처럼 보일 수 있지만,
문제는 그중 **왼쪽에서 가장 먼저 나오는 문자**를 반환해야 하므로 정답은 `"w"`입니다.

가장 간단한 방법은 두 번 순회하는 것입니다.

1. 첫 번째 순회에서 각 문자의 등장 횟수를 셉니다.
2. 두 번째 순회에서 문자열을 다시 처음부터 보며, 등장 횟수가 `1`인 문자를 찾습니다.
3. 처음 발견한 그 문자를 바로 반환합니다.
4. 끝까지 못 찾으면 빈 문자열 `""`을 반환합니다.

이 방법이 좋은 이유는 다음과 같습니다.

- 등장 횟수 판단은 빠르게 할 수 있고
- 원래 문자열 순서를 그대로 다시 확인할 수 있어서
- “처음으로 한 번만 등장한 문자”라는 조건을 정확히 만족할 수 있습니다.

예를 들어 `"aabbccd"`는 마지막 `d`만 한 번 등장하므로 `"d"`를 반환합니다.
반대로 `"aabbcc"`는 모든 문자가 두 번씩 등장하므로 정답은 빈 문자열입니다.

시간 복잡도는 문자열을 두 번 보는 `O(n)`이고, 추가 공간은 문자 개수를 저장하는 데 사용됩니다.
